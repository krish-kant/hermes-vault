# Welcome to your Second Brain

This vault is connected directly to **Hermes**. 
Because it is mounted inside `/opt/data/workspaces/obsidian` in the container, Hermes can read, write, and search any markdown file in this directory.

### The P.A.R.A. Method
We use a standard organization structure:
- **00-Inbox**: Where Hermes drops new research or raw notes.
- **01-Projects**: Notes for active work (e.g., `Supercheck`, `Prodline`).
- **02-Areas**: Ongoing responsibilities.
- **03-Resources**: Code snippets, API documentation, reference material.
- **04-Archive**: Completed projects and old notes.
